We can make apps easily using google colab. The steps are in bellow.



STEP-1 -->
  1. At frist you need to go to the official website of Google Colab

  2. Then Login with your Google account. Then close the browser.

  3. Now go to the same website of Google Colab. Press on the new notebook.

  4. Now copy the commands step by step and keep executing one by one.

    5. !sudo apt update
    !sudo apt install -y git zip unzip openjdk-17-jdk python3-pip autoconf libtool pkg-config zlib1g-dev libncurses5-dev libncursesw5-dev libtinfo5 cmake libffi-dev libssl-dev
    !pip3 install --user --upgrade Cython==0.29.33 virtualenv
    !git clone https://github.com/kivy/buildozer
    %cd buildozer
    !python setup.py build
    !pip install -e .
    %cd ..

  5. Now upload your main.py (main python file) file and the other needed items for the app.

  6. Then run the command (available in the bellow) and press 'y' + 'Enter' on your keyboard if requries.

    7. !buildozer init

  7. Now you can see that a new file named "builozer.spec" file is created.
    Double click on the file to edit. In here you need to do some changes.


    Changes in buildozer.spec file


    • App Name
    • App Domain
    • Package Name
    • Requrements

      This is most important because most of the devloper fails to make apk for it.
      You need to specifi the module and libries used in your application . you have to maintion the version also which you are using in your device.

    • Permission(if requries)
    • Icon path
    • Splash screen path

  8. Now run the Last commond. And wait for 15-20 minutes until fully execution.

    9. !buildozer -v android debug

  9. Now you can find your apk in the bin folder of the file browser.

  10. Right click in it and download.

  11. Enjoy your frist app.
